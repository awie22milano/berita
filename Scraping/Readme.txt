Cara Melakukan Scraping Web Menggunakan Python 3.6 :
1. Install Library-library Python yang dibutuhkan
   - pip install beautifulsoup4
   - pip install selenium
   - pip install lxml
2. Pastikan crhomedrive.exe ada di folder yang sama dengan chr_driver.py
3. Jalankan chr_driver.py
4. Jalankan scraping_google_news.py
5. Akan menghasilkan file berita.csv
6. Import file berita.csv ke tabel berita
